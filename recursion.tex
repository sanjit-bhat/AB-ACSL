Problem 1. Find j(15). Given:


$$
j(x) = \left\{
        \begin{array}{ll}
            2[j(x-3) + 2] & \quad if x >= 10 \\
            j(x-2) + 3 & \quad if 8 <= x < 10 \\
            x & \quad if x < 8
        \end{array}
    \right.
$$

Should be a straightforward problem. Just recurse through the cases.

\[j(15) -> 2[j(12) + 2] = \]
\[j(12) -> 2[j(9) + 2] = \]
\[j(9) -> j(7) + 3 = \]
\[j(7)  = 7\]
\begin{center}
    Derive to bottom case and work back up
\end{center}
\[j(9)  = 10\]
\[j(12)  = 24\]
\[j(15)  = 52\]
\[52\]
